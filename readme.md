# windbell 风铃 轻代码开发平台

## 介绍
风铃平台是新一代面向新移动化的基础开发平台,专门为开发者提供面向钉钉,企业微信,自己构建小程序,自己构建app以及开发平台的基础平台
无论是个人、团队、或是企业，都能够用风铃快速开发上线。网站请看 [https://opensource.afterturn.cn/](https://opensource.afterturn.cn/)
